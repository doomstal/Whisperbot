This is a simple script written by gnurfk, which routes public chat to another character. 
It has a few useful features, such as spam protection, auto response to whispers, etc. 
You need Python version 2.7 to run this.
Simply edit the relevant fields in the script with your favorite text editor, 
park your character where you want it to send and receive chat, then run the script.
whisperbot.py is for logging into servers with no select world dialogue.
whisperbot2 is for servers like the main that have a choice. it chooses the first one, by default.
